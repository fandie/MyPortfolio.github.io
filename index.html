About Me

Saya adalah seorang web programmer, ilmuwan data, dan social media Specialist. Dengan kecintaan terhadap teknologi, saya berkomitmen untuk terus belajar dan berkembang agar saya bisa menghasilkan karya dan inovasi.

Dengan latar belakang pendidikan dan pekerjaan terakhir saya, saya memiliki beberapa keahlian diantaranya sebagai :

  • Front-end web developer dapat mengembangkan aplikasi web yang responsif untuk berbagai perangkat dan menginplementasikan desain web kedalam kode HTML, CSS dan Javascript.
  • Data Scientist mampu dalam mengolah data dengan cara mengeksplorasi, menganalisis dan juga melakukan visualisasi data untuk mendapatkan wawasan yang bernilai untuk bisnis.
  • Social Media Specialist dapat membuat ide konten yang menarik, mengelola dan mengoptimalkan profil perusahaan di platform social media, serta menganalisa tren untuk mengukur kinerja kampanye.
